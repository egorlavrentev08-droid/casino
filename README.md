# Casino Bot

🎰 Telegram бот для игры в рулетку, мины и джокера с системой золота и аукциона.

## Команды
- `/start` - Главное меню
- `/balance` - Баланс
- `/bonus` - Ежедневный бонус
- `/pay [сумма]` - Перевод денег (ответом)
- `/roulette [ставка] [сумма]` - Рулетка
- `/mines [мин] [ставка]` - Мины
- `/joker [ставка]` - Джокер
- `/gold check` - Цена золота
- `/gold my` - Моё золото
- `/gold auc` - Аукцион
- `/anon` / `/deanon` - Анонимность

## Админ команды
- `/balic` - Общая сумма
- `/players` - Статистика игроков
- `/top [n]` - Топ богачей
- `/give [id] [сумма]` - Выдать
- `/take [id] [сумма]` - Забрать
- `/reset [id]` - Сброс игрока
- `/log [id]` - Логи игрока
- `/resetinactive [день/неделя/месяц]` - Сброс неактивных
